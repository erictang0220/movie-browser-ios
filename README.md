# movie-browser-ios

iOS app for browsing movies now playing in theaters, built with Swift and the [The Movie Database API](http://docs.themoviedb.apiary.io/#). Built as part of CodePath's iOS development course.

---

## Part 2

### Features

**Required**
- [x] User can tap a cell to see more details about a particular movie
- [x] User can tap a tab bar button to view a grid layout of Movie Posters using a CollectionView

**Bonus**
- [x] User can tap a poster in the collection view to see a detail screen
- [x] In the detail view, user can tap the poster to view the trailer modally
